# Brain Games
Console mini games
## For user
### How to install
 - install Node.js https://nodejs.org/
 - run `npm install mrixs-brain-games`
### How to run
 - run `brain-games`
## For developer
### How to install
 - install Node.js https://nodejs.org/
 - clone this repository
 - run `make install`
### How to run
 - run `make start`
